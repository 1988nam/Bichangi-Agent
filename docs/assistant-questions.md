# Assistant Question Graph

Answer in Korean or English. Short answers are fine.

## Node 1: ask_purpose

What should this assistant do first?

Examples:
- Daily schedule and task planning
- Email/document drafting
- Research summaries
- Coding project helper
- Personal diary and reflection assistant

## Node 2: ask_audience

Who is the assistant serving, and what tone should it use?

Examples:
- Only me, casual Korean
- Work team, concise professional Korean
- Customers, polished and careful

## Node 3: ask_tools

Which tools should it be able to use?

Examples:
- Local files
- Calendar
- Email
- Web search
- Browser automation
- GitHub
- Notion or Google Docs

## Node 4: ask_memory

What should it remember over time?

Examples:
- Preferences and writing style
- Project history
- People and relationships
- Daily routines
- Nothing unless I approve it

## Node 5: ask_interface

Where should you use it?

Examples:
- Terminal CLI
- Local web app
- Codex thread workflow
- ChatGPT app or MCP tool
- Mobile-friendly web page

## Node 6: ask_safety

What actions require confirmation?

Examples:
- Sending email
- Editing or deleting files
- Purchasing or booking
- Posting online
- Running commands

## Output: build_brief

The graph will summarize:
- Assistant mission
- Primary workflows
- Tools and permissions
- Memory policy
- Interface choice
- Confirmation rules
